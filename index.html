Múltiples Dimensiones

Resolución de problemas complejos de transferencia de calor mediante la superposición de soluciones unidimensionales.

1 El Salto a las 3 Dimensiones

En la práctica, muchos cuerpos como bloques rectangulares o cilindros cortos presentan gradientes de temperatura en más de una dirección. Para estos casos, la temperatura es una función de varias coordenadas espaciales y del tiempo: $T(x, y, z, t)$.

"La solución para una geometría multidimensional puede expresarse como el producto de las soluciones de geometrías unidimensionales componentes." [Cengel, p. 11]

¿Por qué usar este método?

  • Evita resolver ecuaciones diferenciales parciales complejas.
  • Permite usar las gráficas de Heisler o soluciones de un término.
  • Es exacto para cuerpos con temperatura inicial uniforme y condiciones de frontera constantes.

2. Teorema del Producto

Definimos la temperatura adimensional $\theta(x, t)$ para el caso general como:

$$\theta(x, y, z, t) = \frac{T(x, y, z, t) - T_\infty}{T_i - T_\infty}$$
Fórmula Maestra de Superposición

Bloque Rectangular (3D)

Es la intersección de 3 paredes planas infinitas de espesores $2L_1, 2L_2, 2L_3$:

$$\theta_{rect}(x,y,z,t) = \theta_{pared}(x,t) \cdot \theta_{pared}(y,t) \cdot \theta_{pared}(z,t)$$

Cilindro Corto (2D)

Es la intersección de un cilindro infinito de radio $r_o$ y una pared plana de espesor $2L$:

$$\theta_{cil\_corto}(r,z,t) = \theta_{inf\_cil}(r,t) \cdot \theta_{pared}(z,t)$$

4. Ejemplo de Aplicación

Enfriamiento de un Cilindro Corto de Aluminio

Problema típico de examen - Uso del Teorema del Producto

Enunciado: Un cilindro de aluminio ($k=237 W/m\cdot K, \alpha=9.71\times 10^{-5} m^2/s$) de $10 \text{ cm}$ de altura y $15 \text{ cm}$ de diámetro se extrae de un horno a $400^\circ C$ y se sumerge en agua a $20^\circ C$ ($h=120 W/m^2\cdot K$). Calcule la temperatura en el centro tras $15$ minutos.

Datos:

  • $T_i = 400^\circ C$
  • $T_\infty = 20^\circ C$
  • $t = 900 \text{ s}$
  • $L = 0.05 \text{ m}$ (mitad altura)
  • $r_o = 0.075 \text{ m}$ (radio)

// PASO 1: Calcular Bi y Fo para pared y cilindro

Pared: $Bi = 0.025, Fo = 35 \implies \theta_{pared} \approx 0.42$

Cilindro Inf: $Bi = 0.038, Fo = 15.5 \implies \theta_{inf\_cil} \approx 0.31$

// PASO 2: Aplicar Teorema del Producto

$\theta_{total} = \theta_{pared} \cdot \theta_{inf\_cil}$

$\theta_{total} = 0.42 \times 0.31 = 0.1302$

// PASO 3: Convertir a Temperatura

$T = T_\infty + \theta_{total}(T_i - T_\infty)$

$T = 20 + 0.1302(380) = 69.4^\circ C$

Resultado: 69.4 °C
